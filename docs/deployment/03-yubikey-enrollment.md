# YubiKey Enrollment (FIDO2 / Passkey)

## Steps

1. Navigate to:
   https://mysignins.microsoft.com/security-info

2. Click:
   Add sign-in method

3. Select:
   Passkey

4. Complete MFA (Authenticator)

5. Insert YubiKey

6. Create PIN
   - Stored on device
   - Minimum 4 digits (longer recommended)

7. Touch the key

8. Name the key

---

## Result

- YubiKey registered
- Authenticator remains fallback (lab phase)
