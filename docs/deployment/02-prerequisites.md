# Prerequisites

## Identity Requirements
- Microsoft Entra ID tenant
- Global Administrator access

## Accounts
- 1 active admin (test account)
- 1 break-glass account (excluded from CA)

## Authentication Methods
Enable:
- Passkeys (FIDO2)
- Microsoft Authenticator
- Temporary Access Pass (recommended)

## Hardware
- YubiKey with FIDO2 support
- Backup key recommended

## Client Requirements
- Microsoft Edge (recommended)
- Chrome supported
